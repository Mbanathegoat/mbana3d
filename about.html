About This Project

Welcome to the Coca Cola Branded 3D App, a project designed as part of my coursework for Advancved Computer Scienece at the University of Sussex. This project aims to integrate 3D models into a web-based application using modern web technologies and principles, such as HTML5, CSS3, JavaScript, and PHP. The application adheres to the Model-View-Controller (MVC) architecture, ensuring a clear separation of concerns and enhancing maintainability.

The 3D App features a selection of Coca Cola branded products, each rendered in an interactive 3D space. Users can explore and manipulate these models using a variety of tools and options available in the app's interface. The project leverages X3DOM for rendering the 3D models directly within the web browser, providing a seamless and immersive user experience.

Key Features

  • Interactive 3D models of Coca Cola products, including cans and bottles.
  • Responsive user interface designed using Bootstrap 4, ensuring compatibility across devices.
  • Dynamic content loading and interaction powered by JavaScript, jQuery, AJAX, and JSON.
  • Data management and storage handled by a SQLite database, accessed via PHP scripts.
  • Implementation of the MVC design pattern for organized and efficient code structure.
  • Detailed object view implemented using glightbox

Technologies Used

  • HTML5 & CSS3: For structuring and styling the web pages.
  • JavaScript & jQuery: To handle interactivity and DOM manipulation.
  • X3DOM: For embedding and controlling 3D content in the browser.
  • PHP: For server-side logic and database interaction.
  • SQLite: A lightweight database solution for storing model data.
  • AJAX & JSON: For asynchronous data exchange between the frontend and backend.

Project Goals

The primary goal of this project is to demonstrate proficiency in building web applications that integrate 3D content and interactivity using modern web development techniques. By adhering to the MVC pattern, the project also aims to showcase the importance of organized code architecture in building scalable and maintainable web applications.

This project serves as a comprehensive learning experience, combining different aspects of web development, from frontend design and interaction to backend logic and database management.

Submitted in partial fulfilment of the Advanced Computer Science Masters Degree at the University of Sussex
